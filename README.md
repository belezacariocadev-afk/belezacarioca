# Correção tela cheia - Beleza Carioca

Suba estes arquivos no repositório substituindo os existentes:

- app/page.tsx
- app/layout.tsx
- app/globals.css

Depois rode:

```bash
npm run build
```

Se o projeto já tiver conteúdo importante em `layout.tsx` ou `globals.css`, não apague sem conferir. Nesse caso, copie principalmente as regras de CSS de tela cheia.
